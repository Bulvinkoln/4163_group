Работа с удаленными репозиториями:

Чтобы начать работать с удаленными репозиториями, надо установить/зарегистрироаться в GitHub.
GitHub - это сервер, где мы можем сохранять наши работы для того, чтобы дать доступ другим пользователям. Также мы можем работать в одном документы на разных устройствах, например дом/работа, что позволяет получить доступ к данному материалу без каких-либо проблем.

Для работы с удаленным репозиторием мы используем следующие команды:

git clone - копирует репозиторий на локальный компьютер следующим образом: git clone "ссылка на репозиторий"
git pull - скачиваем данные е с репозитория GitHub на локальный компьютер: git pull origin "название ветки"
git push - мы работаем в скаченном файле и отправляем в GitHub:git push -u origin "название папки"
pull request - используется для того, чтобы ты мог предложить свой вариант работы/ внесение дополнений в работу другого пользователя. 

Pull Request выполняется в следующем порядке: 

Fork -> git clone(своей версии) -> git branch (добавляем ветку) -> git add ->git commit ->git push -u origin "название ветки" -> compare & pull request.
