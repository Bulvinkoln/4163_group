Одна из инструкций по работе с репозиториями:

Чтобы удалить последний commit в Git, можно использовать команду git reset. Эта команда позволяет переместить указатель текущей ветки на определенный коммит.

Откройте терминал и перейдите в директорию проекта Git.
Введите команду git log чтобы убедиться, что последний commit имеет нужный хеш-код. Скопируйте этот хеш-код.
Введите команду git reset --hard HEAD~1, где HEAD~1 означает последний коммит.
Введите команду git push --force, чтобы загрузить изменения на удаленный репозиторий.
Важно понимать, что использование git reset может привести к потере данных, если коммит был опубликован на удаленном репозитории и к нему были присоединены другие коммиты. Поэтому будьте осторожны при использовании этой команды и убедитесь, что вы понимаете, что делаете.
